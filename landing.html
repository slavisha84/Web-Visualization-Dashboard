Summary: Latitude vs. X

The goal in this assignment is to visualy present wether anlysis of 500+ cities across the world with their respect to distance from equator. We will create severl of scatter plots to showcase relationships such as:Temperature (F) vs. Latitude, Humidity (%) vs. Latitude, Cloudiness (%) vs. Latitude, Wind Speed (mph) vs. Latitude. In order to complete this we created a Python script that collected data from OpenWeatherMap API to create a representative model of weather across world cities.
